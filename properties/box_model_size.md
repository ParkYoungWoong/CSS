## `width`

요소의 '가로 너비'를 지정

| 값 | 의미 | 기본값 |
|---|---|---|
| `auto` | 브라우저가 너비를 계산 | `auto` |
| 단위 | `px`, `cm` 등 단위로 지정 |  |

```css
.block {
  display: block;
  width: auto; /* 100% 동등 */
}
.inline {
  display: inline;
  width: auto; /* 0 동등 */
}
```

### `max-width`

요소가 커질 수 있는 '최대 가로 너비' 지정

| 값 | 의미 | 기본값 |
|---|---|---|
| `none` | 최대 가로 너비 없음 | `none` |
| 단위 | `px`, `cm` 등 단위로 지정 |  |

### `min-width`

요소가 작아질 수 있는 '최소 가로 너비' 지정

| 값 | 의미 | 기본값 |
